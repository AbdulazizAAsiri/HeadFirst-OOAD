====== Synchronization ======
Created Saturday 27 February 2016

**MySQL Workbench - Model To Localhost Database**
To synchronize changes from the model to the database, do the following:
**Database > Synchronize With Any Source**
Here, use the following settings:
	1. **Source**: Model Schemata
	2. **Destination**: Live Database Server
	3. **Send Updates To**:
		(For File)   Any script file to run at:
				C:\Users\Mark\Documents\Projects\Website\MarkPThomas-Documentation\mySQL EERs\Scripts - Schemas
		(For updating [[MySQL]] directly)
				(Destination Database Server)
In **Target Database**, set stored connection to: Local instance MySQL56

**Localhost Database to MySQLWorkbench Model**
To synchornize changes from the database to the model, do the following:


**LocalHost Database to File (Including Data)**
**Server > Data Export**
	1. Select database object (mountaineering)
	2. **Options > Export to Self-Contained File**
		a. Also check '**Create Dump in a Single Transaction**' if all tables should be in a single file.
